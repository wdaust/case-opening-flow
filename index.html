⚠ OA Quality Gate — Cannot mark OA complete without:

  • Preferred contact method + best time window
  • Provider list (at least first treating provider or "none yet")
  • Medicare / Medicaid / ERISA / VA flags (Yes / No / Unknown)
  • Evidence inventory status (Complete or Pending)
  • Prior injuries flag — if YES: attorney notified within 24 hrs
  • Treatment gaps flag — if YES: attorney notified within 24 hrs

Case Opening Complete — Final Gate
Gate Check System Validation ▾

All boxes checked = case status changes to Active. Eligible for demand prep when treatment completes.

⚠ Case Opening is COMPLETE only when ALL are true:

  • Engagement letter signed (DocuSign)
  • All 5 scoring systems completed (all score fields populated)
  • Data Hygiene Gate passed (all required fields complete)
  • Complaint filed (court receipt on file)
  • Service initiated (summons + process server documented)
  • Medical records requested (all known providers have active requests)
  • Post-OA automation burst confirmed (all 6 auto tasks verified)
  • OPRA / evidence requests sent (tasks marked complete)
✓ Case Status → Active
Now eligible for demand preparation when treatment completes.
